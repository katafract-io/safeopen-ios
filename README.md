# SafeOpen

Privacy-first QR and link inspection. Katafract family.

**What it does:** Scan QR codes or paste links. Before you open anything, SafeOpen tells you what it is, what it will do, and whether it's risky.

**What it is not:** A VPN. A browser. A phishing research console.

See `SafeOpen/Docs/` for architecture, roadmap, and agent instructions.
